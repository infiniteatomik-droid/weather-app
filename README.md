# 📋 Интерактивная Канбан-доска (Практический проект)

### 🔗 [👉 Живое демо проекта (Посмотреть вживую) 👈](СЮДА_ВСТАВЬ_СВОЮ_ССЫЛКУ)

Простая и удобная Канбан-доска для управления задачами прямо в браузере. Проект создан в рамках изучения основ JavaScript, работы с веб-хранилищем и Drag-and-Drop API.

---

## ✨ Основные функции

* **Добавление задач:** Быстрое создание карточек планов через форму ввода.
* **Перетаскивание (Drag and Drop):** Удобное перемещение карточек между колонками «Планы», «В работе» и «Готово» с помощью мышки.
* **Клик-навигация:** Альтернативный способ перемещения карточек по этапам (и их удаления) по обычному клику.
* **Сохранение данных (Local Storage):** Все задачи автоматически записываются в память браузера. При перезагрузке страницы или закрытии вкладки ваши данные не пропадают.

---

## 🛠 Технологический стек

* **HTML5:** Семантическая структура приложения.
* **CSS3:** Адаптивная верстка, сетка Flexbox, плавные анимации при наведении (`hover`).
* **Vanilla JavaScript:** Чистый JS без использования сторонних библиотек и фреймворков.
* **Браузерные API:** 
  * `HTML5 Drag and Drop API` (события `dragstart`, `dragover`, `drop`)
  * `Web Storage API` (`localStorage`)

---

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий или скачайте архив с кодом:
   ```bash
   git clone https://github.com
   ```
2. Откройте папку с проектом.
3. Дважды кликните по файлу `index.html`, чтобы открыть приложение в любом современном браузере.

---

## 💡 Что было изучено и реализовано в ходе проекта:

1. **Единственный источник правды:** Вся логика завязана на один массив объектов `tasks`, а экран (`renderTasks`) подстраивается под актуальные данные.
2. **Сериализация данных:** Изучена работа с JSON (`JSON.stringify` для упаковки в строку и `JSON.parse` для распаковки массива).
3. **Безопасная загрузка:** Реализована проверка памяти при старте страницы с помощью тернарного оператора, защищающая приложение от ошибок при первом запуске.
4. **Оптимизация событий:** События сброса (`drop`) вынесены за пределы циклов, что предотвращает утечки памяти и лаги интерфейса.
